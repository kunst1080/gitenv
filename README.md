gitenv
===

A .gitconfig switcher.

# Usage

```
$ gitenv --help
NAME:
  gitenv - A .gitconfig switcher.

OPTIONS:
  -s, --show (default)
    Display the current config.
  -l, --list
    List configs.
  -c, --config CONFIG_NAME
    Create a config and change to the config.
  --rm CONFIG_NAME
    Remove the config.
  -h, --help
    Display this help.
```

# Installation

Copy the `gitenv` script to your bin directory.
